# CV Directory

This directory contains the downloadable CV files for the portfolio website.

## Required Files

Place your CV files in this directory with the following names:

- `cv-web-developer.pdf` - Your web development CV
- `cv-video-editor.pdf` - Your video editing CV

## Creating Your CVs

1. Create two separate CVs tailored for each role (Web Developer and Video Editor)
2. Export them as PDF files
3. Name them exactly as specified above
4. Place them in this directory

The portfolio website will automatically link to these files on the CV download page.
