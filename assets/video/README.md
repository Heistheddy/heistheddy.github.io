# Video Project Assets

Place your video project thumbnail images in this directory.

## Required Images

For each project in `/data/video-projects.json`, you'll need:

- Thumbnail image (e.g., `commercial-thumb.jpg`)

## Image Guidelines

- Use 16:9 aspect ratio thumbnails (1280x720 or 1920x1080)
- Create eye-catching thumbnails that represent your video content
- Optimize images for web (compress to reduce file size)
- Use consistent naming convention matching your JSON data
- Supported formats: JPG, PNG, WebP

## Example Structure

```
video/
  commercial-thumb.jpg
  iceland-thumb.jpg
  music-video-thumb.jpg
  corporate-thumb.jpg
  documentary-thumb.jpg
  product-demo-thumb.jpg
```
