# Web Project Assets

Place your web development project images in this directory.

## Required Images

For each project in `/data/web-projects.json`, you'll need:

- Thumbnail image (e.g., `ecommerce-thumb.jpg`)
- Screenshot images (e.g., `ecommerce-1.jpg`, `ecommerce-2.jpg`)

## Image Guidelines

- Use high-quality screenshots (1920x1080 or higher)
- Optimize images for web (compress to reduce file size)
- Use consistent naming convention matching your JSON data
- Supported formats: JPG, PNG, WebP

## Example Structure

```
web/
  ecommerce-thumb.jpg
  ecommerce-1.jpg
  ecommerce-2.jpg
  taskapp-thumb.jpg
  taskapp-1.jpg
  taskapp-2.jpg
```
